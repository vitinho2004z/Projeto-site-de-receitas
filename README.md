# Projeto-site-de-Sobremesas-
Um site simples e intuitivo de receitas de sobremesas, criado para quem ama cozinhar sem complicação. Aqui você encontra pratos deliciosos, com passo a passo fácil, ingredientes acessíveis e um visual limpo que torna a experiência leve e agradável. Ideal tanto para iniciantes quanto para quem quer explorar novos sabores no dia a dia.

Alunos:Hamilton Felipe, Caio Guilherme, João Vitor C. Queiroz

5W3H

    Desenvolvimento de um site simples e intuitivo de receitas de sobremesas, com passo a passo fácil, ingredientes acessíveis e visual agradável.
 2. QUEM?
    Alunos: Hamilton Felipe, Caio Guilherme e João Vitor C. Queiroz.
